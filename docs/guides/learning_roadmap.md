## 游戏方向：殖民地模拟 / 经营类

### 兴趣定位

| 感兴趣 | 不感兴趣 |
|--------|---------|
| 世界模拟 | 图形学/渲染管线 |
| 经济活动 | Shader 编写 |
| 互动结构 / 涌现行为 | 3D 技术 |
| 系统设计 | SIMD 优化 |

### 参考游戏

| 游戏 | 核心系统 | 渲染复杂度 |
|------|---------|-----------|
| RimWorld | AI 行为树 + 需求 + 关系 + 故事生成 | 简单 2D |
| Dwarf Fortress | 世界历史 + 流体 + 材料 + AI | 长期只有 ASCII |
| Factorio | 物流网络 + 电力 + 流体管道 | 简单 2D |
| Kenshi | 开放世界 + 经济 + 派系 | 简单 3D |

共同点：**渲染都很简单，核心是模拟。**

### 技术架构

┌─────────────────────────────────────────┐

## 开发计划

### 第一阶段：引擎基础（补完 HH）

```
第 1-2 周    物理+碰撞（HH Day 041-050）
             - 运动方程：Delta = 0.5*ddP*dt² + dP*dt
             - Minkowski 碰撞检测
             - 碰撞响应（滑动）

第 3 周      SimRegion 完善（HH Day 051-065）
             - end_sim 回写 + chunk 迁移
             - 多 chunk 加载
             - 相机跟随
```

### 第二阶段：模拟游戏核心系统

```
第 4 周      A* 寻路
             - 网格地图 + 障碍物
             - 殖民者自动寻路到目标

第 5 周      工作系统
             - 行为状态机：Idle → MoveTo → DoWork → Idle
             - 砍树、搬运、建造

第 6 周      需求系统
             - 饥饿、困倦、心情
             - 需求驱动行为优先级

第 7 周      建造系统
             - 玩家下达建造指令
             - 殖民者自动搬材料、建造

第 8 周      故事生成器
             - 定时触发事件（袭击、天气、旅行者）
             - 根据殖民地状态调整难度
```

### 第三阶段：打磨和扩展

```
存档系统     序列化 GameState 到文件
UI 系统      Raylib DrawText + 简单面板
经济系统     资源、交易、供需
更多实体     动物、敌人、NPC 商人
```

## 关键原则

| 原则 | 说明 |
|------|------|
| **做游戏，缺什么学什么** | 不要预先学完所有系统 |
| **涌现 > 脚本** | 用简单规则碰撞出复杂行为 |
| **渲染够用就行** | 方块就能当殖民者 |
| **每步都可运行** | 每周都有能玩的东西 |
| **系统间解耦** | 需求系统不依赖渲染，寻路不依赖 AI |
